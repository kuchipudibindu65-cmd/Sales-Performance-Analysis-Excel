This project analyzes sales data to identify top-performing products and regions using Excel Pivot Tables and Dashboards.

The dataset contains sales information such as product, category, quantity, price, region, and customer type. Revenue was calculated and used for analysis.

Key analyses performed:
• Product-wise revenue analysis
• Region-wise revenue analysis
• Daily and monthly sales trends
• Interactive dashboard using Pivot Charts

Tools Used:
• Microsoft Excel
• Pivot Tables
• Pivot Charts
• Data Cleaning
• Data Analysis

Key Insights:
• Water Bottle is the highest revenue generating product
• East region contributes the highest sales
• Total revenue for the period is ₹15,150
